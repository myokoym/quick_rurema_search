# Quickるりまサーチ

迅速に[るりまサーチ][]を検索できるGoogle Chrome拡張機能です。

[るりまサーチ]: http://docs.ruby-lang.org/ja/search/

## 使い方

1. テキストを選択
2. 右クリックメニューから「Quickるりまサーチ」を選択
3. るりまサーチの検索結果が新しいタブで表示される
